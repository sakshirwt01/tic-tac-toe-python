# 🎮 Tic Tac Toe Game (Console-Based Python Project)

## 📌 Project Overview
This is a **console-based Tic Tac Toe game** developed using Python.  
The game allows **two players** to play interactively in the terminal by taking turns and marking their positions on a 3×3 grid.

This project focuses on building strong fundamentals in **control statements, loops, and basic game logic**, making it ideal for beginners.

## 🚀 Features
- 🎯 Two-player interactive gameplay (Player X & Player O)
- 🔁 Continuous game loop using `while`
- ✅ Input validation (prevents invalid or repeated moves)
- 🧠 Win detection logic (rows, columns, diagonals)
- 🤝 Draw detection when board is full
- 🖥️ Clean and simple console-based UI

## 🛠️ Technologies Used
- **Python**
- Control Statements (`if-elif-else`)
- Loops (`while`)
- Lists (for board representation)

## 🎲 How the Game Works
1. The game board consists of 9 positions (3×3 grid)
2. Players take turns choosing positions (1–9)
3. The board updates after each move
4. The program checks:
   - If a player has won
   - If the game is a draw
5. The game ends when:
   - A player wins 🎉  
   - OR the board is full (Draw)

## ▶️ How to Run the Project

1. Clone the repository
2. Run the file:
   python main.py.
